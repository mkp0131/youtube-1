# github 회원가입 / 로그인 flow

1. 원하는정보를 포함한 url 로 1단계 요청을 보낸다.
2. 1단계 요청이 완료되면 github 에서 callback url + 코드값 로 리다이렉션 한다.
3. callback url 에서 코드값 을 활요하여 유저 정보를 얻는 요청을 보낸다.
4. 만약 유저테이블에 이메일이 등록이 되어있다면 로그인처리, 없다면 회원가입 처리한다.
